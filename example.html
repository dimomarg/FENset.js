fenSet.js example document

Make sure you have OpenChessFont.otf installed!

Using fenSet.js, FEN strings are converted to diagrams for use with chess symbol fonts on page load.

Simply put your positions in divs with class "fen" and use CSS to style them.

rnbqkbnr/pp1ppppp/8/2p5/4P3/5N2/PPPP1PPP/RNBQKB1R b KQkq - 1 2

To make a diagram from black's perspective, also put "flipped" in the div's class.

rnbqkbnr/pp1ppppp/8/2p5/4P3/5N2/PPPP1PPP/RNBQKB1R b KQkq - 1 2

For white characters on black backgrounds, use the "darkmode" class

rnbqkbnr/pp1ppppp/8/2p5/4P3/5N2/PPPP1PPP/RNBQKB1R b KQkq - 1 2

If your chess font uses different letters for each glyph, change fenSet.symbolMap accrordingly.

fenSet.js also works with Vivliostyle.
